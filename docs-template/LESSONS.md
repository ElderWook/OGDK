# LESSONS — the learning loop's capture buffer

> Append-only. Every session that hits **friction the system didn't prevent** logs it
> here before ending (session-end skill, step 4). Entries are raw material for the
> `kit-retro` skill, which periodically converts them into permanent upgrades to
> skills, AGENTS rules, gate scripts, and reference docs — then marks them CODIFIED.
>
> Log it if: a rule was unclear, a script gave a wrong/confusing answer, a model made
> the same mistake twice, a manual step should have been mechanical, a hazard was
> survived by luck. Don't log: one-off typos, things the gate already caught cleanly.

Format (append at the bottom):

```
## YYYY-MM-DD <short title>
**What happened:** one or two sentences.
**Root cause:** why the system allowed it.
**Proposed fix:** rule / skill edit / script change / reference page.
**Status:** OPEN | CODIFIED <date, where>
```
